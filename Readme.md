## 环境变量

### 数据库

| 变量名         | 说明     | 默认值        | 可选值                     |
|-------------|--------|------------|-------------------------|
| DB_ENGINE   | 数据库引擎  | postgresql | postgresql,mysql,sqlite |
| DB_NAME     | 数据库名   |            |                         |
| DB_USER     | 数据库用户名 |            |                         |
| DB_PASSWORD | 数据库密码  |            |                         |
| DB_HOST     | 数据库地址  |            |                         |
| DB_PORT     | 数据库端口  |            |                         |
